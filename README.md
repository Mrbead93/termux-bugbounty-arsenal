# 🛡️ Termux Bug Bounty Arsenal (Unrooted)

**The first complete all-in-one bug bounty toolkit built exclusively for unrooted Android phones.**

Tested on **Samsung Z Fold 7** • No root required • Optimized for mobile hunting.

---

## ✨ Features

- Direct integration with **ReconX** and **CredX**
- Full automated bug bounty chain (covers recon → scanning → fuzzing)
- Beautiful progress bar + spinner during long scans
- Professional Markdown report generator (ready for submission)
- Mobile-first design
